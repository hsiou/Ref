# Security Policy

## Supported versions

Security fixes are applied to the latest released version on `main`.

## Reporting a vulnerability

Please do not open a public issue for suspected vulnerabilities.

Use GitHub private advisories:

- https://github.com/drpedapati/pubmed-cli/security/advisories/new

Include:

1. Affected version or commit.
2. Reproduction steps.
3. Impact assessment.
4. Suggested remediation (if known).
